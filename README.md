# Countdown Timer App

A Countdown Timer application using React Native

## Installation

```bash
$ yarn install
```

## iOS Installation

```bash
$ cd ios && pod install
```

## Running the app

```bash
# iOS devices
$ yarn ios

# Android devices
$ yarn android
```

## Test

```bash
# unit tests
$ yarn test
```
